难点：
    1. 创建了一个处理编译器进程函数，这个是干什么的？
        创建实际的编译过程，它是一个结构
    2. 创建了一个helpers的文件夹，这个是用来干什么的？
    3. malloc函数和calloc函数的区别有两个
        malloc要自己计算字节数，calloc不需要
        calloc会所有字节初始化为0，很方便，但效率也更低
    4. realloc函数可用于堆空间扩容，会将原空间的内容复制到新空间，并且自动free原空间，是一个很方便的函数
    5. 为什么要进行词法分析？
        更方便理解源码，快速识别源码中的关键字，符号，操作符，变量常量
        不去扫描源码，更简单
        数字在源码中是字符串，此法分析将其转换成整型或其他数字类型，更方便
        忽略了空格，制表符这些不必要信息
    6. 此法分析的类型
        标识符，关键字，操作符，分隔符，数字，字符串，注释，新行

    8. token结构里的东西很多都不知道是用来干什么的。
